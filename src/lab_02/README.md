# Лабораторная работа №2: Коллекция книг (Библиотека)

## Цель работы
* Научиться работать с коллекциями объектов.

* Реализовать собственный контейнерный класс.

* Освоить итерацию, индексацию, сортировку и фильтрацию коллекции.

* Понять разницу между моделью сущности (*класс Book*) и контейнером объектов (*класс Library*).

## Основные идеи
* **Контейнерная ответственность** – класс *Library* управляет группой объектов *Book*, обеспечивая добавление, удаление, поиск, сортировку и фильтрацию.

* **Инкапсуляция коллекции** – внутреннее хранилище *self._items* (список) скрыто, доступ к элементам предоставляется через методы и магические методы.

* **Защита от дубликатов** – при добавлении проверяется, есть ли уже книга с таким же названием, автором и годом (используется *__eq__* класса *Book*).

* **Магические методы** – *__len__* (поддержка len()), __iter__ (поддержка цикла for), *__getitem__* (поддержка индексации [ ]).

* **Сортировка** – метод *sort(key, reverse)* позволяет упорядочивать книги по любому атрибуту (например, по году, названию, автору).

* **Логические операции** – методы *get_available()*, *get_loaned()*, *filter_by_year()* возвращают новые коллекции *Library*, не изменяя исходную.

* **Обработка ошибок** – при попытке добавить объект не типа *Book* или дубликат выбрасываются исключения *TypeError / ValueError*, которые перехватываются в демонстрации.

## Реализованный класс Library
Коллекция книг (библиотека). Хранит объекты *Book* и предоставляет все необходимые методы управления.

## Атрибуты экземпляра (защищённые)
* **_items** – список объектов Book.

## Методы управления коллекцией
* **add(book: Book)** – добавляет книгу. Проверяет тип и отсутствие дубликата (по *__eq__**).

* **remove(book: Book)** – удаляет книгу из коллекции.

* **remove_at(index: int)** – удаляет книгу по индексу (с проверкой границ).

* **get_all()** – возвращает копию списка всех книг.

## Методы поиска
* **find_by_title(title: str)** – возвращает список книг, в названии которых содержится подстрока (регистронезависимо).

* **find_by_author(author: str)** – возвращает список книг, у которых автор содержит подстроку.

* **find_by_year(year: int)** – возвращает список книг с точным годом издания.

## Магические методы
* **__len__()** – возвращает количество книг в коллекции (позволяет использовать *len(library)*).

* **__iter__()** – возвращает итератор по списку книг (позволяет использовать *for book in library*).

* **__getitem__(index)** – обеспечивает доступ по индексу: *library[0], library[-1]*.

## Сортировка
* **sort(key=None, reverse=False)** – сортирует коллекцию на месте по заданному ключу.
Примеры:

    * **library.sort(key=lambda b: b.year)** – по году (от старых к новым).

    * **library.sort(key=lambda b: b.title)**– по названию.

## Логические операции (фильтрация)
* **get_available()** – возвращает новую коллекцию *Library*, содержащую только доступные книги (*is_loaned == False*).

* **get_loaned()** – возвращает новую коллекцию *Library* с выданными книгами.

* **filter_by_year(min_year, max_year)** – возвращает новую коллекцию с книгами, год издания которых находится в указанном диапазоне.
